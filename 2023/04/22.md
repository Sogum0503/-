## leetcode
### 2388. Change Null Values in a Table to the Previous Value
```sql
WITH num AS(
SELECT id
      ,ROW_NUMBER() OVER() num
      ,drink
FROM coffeeshop
),group_id AS(
SELECT id
       ,num
       ,drink
       ,SUM(1-ISNULL(drink)) OVER(order by num ) group_id
FROM num
ORDER BY num
) SELECT id
      ,FIRST_VALUE(drink) OVER(partition by group_id) drink
FROM group_id
ORDER BY num
```
