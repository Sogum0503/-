## 프로그래머스 
### 이름이 있는 동물의 아이디
```SQL
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
```
### 중성화 여부 파악하기
```SQL
SELECT ANIMAL_ID, 
        NAME,
        CASE WHEN SEX_UPON_INTAKE LIKE 'Neutered%' OR SEX_UPON_INTAKE LIKE 'Spayed%' THEN 'O'
        ELSE 'X' 
        END AS "중성화"
FROM ANIMAL_INS 
ORDER BY 1,2,3
```
