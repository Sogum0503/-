### 즐겨찾기가 가장 많은 식당 정보 출력하기
```SQL
WITH TEMP AS(
SELECT FOOD_TYPE,
        REST_ID,
        REST_NAME,
    SUM(FAVORITES) AS FAVORITES,
        RANK() OVER(PARTITION BY FOOD_TYPE ORDER BY SUM(FAVORITES) DESC) AS RNK        
FROM REST_INFO 
GROUP BY FOOD_TYPE, REST_ID, REST_NAME)

SELECT FOOD_TYPE,
        REST_ID,
        REST_NAME,
        FAVORITES
FROM TEMP 
WHERE RNK = 1 
ORDER BY FOOD_TYPE DESC 
```

### 조건별로 분류하여 주문상태 출력하기
```SQL
SELECT ORDER_ID, 
        PRODUCT_ID,
        DATE_FORMAT(OUT_DATE,'%Y-%m-%d') AS OUT_DATE,
        CASE WHEN OUT_DATE <= '2022-05-01' THEN '출고완료'
              WHEN OUT_DATE > '2022-05-01' THEN '출고대기'
              WHEN OUT_DATE IS NULL THEN '출고미정' END AS '출고여부'
FROM FOOD_ORDER
```
