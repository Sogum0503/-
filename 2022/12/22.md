## leetcode
### 1517. Find Users With Valid E-Mails
```
* a-zA-Z : 알파벳 전부
* 0-9 : 숫자 전부
* ^ : 뒤에 있는 무언가로 시작한다는 뜻
* $ : 앞에 있는 무언가로 끝난다는 뜻 
```
```sql
-- 유효한 이메일은 문자로 시작되며 숫자, _, ., -을 포함할 수 있다.
-- 도메인 주소는 @leetcode.com이다. \

SELECT *
FROM Users 
WHERE mail REGEXP('^[a-zA-Z][a-zA-Z0-9_.-]*@leetcode[.]com$')
```
