### solvesql
#### 그룹 별 중복값이 있는지 확인하기
```sql
-- 각 quartet 안에 같은 x 값이 2개 이상인 경우를 찾아, 몇 개의 값이 중복되는지 구하는 쿼리를 작성
SELECT quartet,
      x,
      count(x) AS cnt 
FROM points 
GROUP BY quartet,x
HAVING COUNT(x) >= 2
```

#### 그룹 별 중복값이 있는지 확인하기
```sql
--  가장 비싼 한 끼 식사를 한 고객이 방문한 요일, 시간대, 같이 식사한 일행 수를 구하는 쿼리를 작성
SELECT day,
      time,
      size
FROM tips 
WHERE total_bill = (SELECT MAX(total_bill)
                    FROM tips)
```

#### 그룹 별 중복값이 있는지 확인하기
```sql
-- 와인 품질 지수가 8 이상이고, 알콜 도수가 12도 이하인 레드 와인의 정보를 모두 출력하는 쿼리
SELECT *
FROM wines 
WHERE quality >= 8 
AND alcohol <= 12 
AND color = 'red'
ORDER BY quality DESC, alcohol ASC 

```
